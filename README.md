# GlowGuide-Frontend
Kişiselleştirilmiş cilt bakımı rutinleri ve ürün takibi sunmak için React Native ve Expo ile geliştirilmiş çapraz platform mobil uygulaması. Modern bir kullanıcı arayüzü, Expo Router ile akıcı navigasyon sunar ve veri odaklı bir cilt bakımı asistanı deneyimi sağlamak için Spring Boot REST API ile entegre çalışır.
